Overview of the Rocket Dodge Game Project

This project is a web-based game where players control a rocket to dodge asteroids. The game is designed to showcase a variety of technologies including:

  • Frontend: HTML/CSS and JavaScript for game rendering and controls.
  • Backend: Python with Django for server-side logic.
  • Database: MySQL for storing user scores and game data.
  • AI Bot: An agent that plays the game to achieve high scores.
