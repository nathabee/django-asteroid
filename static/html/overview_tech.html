Technical Overview

The Rocket Dodge Game Project is built using a variety of technologies designed to provide a robust, interactive gaming experience.

Frontend Technologies

The frontend uses HTML, CSS, and JavaScript. It dynamically updates the game state based on user inputs and communicates with the backend to fetch and submit data.

Backend Technologies

The backend is powered by Django, a Python-based framework that provides a robust environment for developing web applications and APIs. The backend handles user authentication, game logic, and high-score submissions.

Database

MySQL is used to store user data, game scores, and other persistent data required for the game's functionality. This choice provides reliable data management and scalability.

AI Bot

The AI bot is designed using Python and leverages basic algorithms to control the rocket. Future improvements may include implementing more sophisticated AI techniques.
